CORE ANALYSIS SCRIPTS:
  02-growth-rates-growth-tools.R
    - Growthtools method for exponential growth rate estimation
    - Uses get.growth.rate() from growthTools package

  09-growth-rates-no-lag.R
    - Growthtools approach without explicit lag-phase separation
    - Includes all time points in exponential fit
    - Output: all-blocks-growth-no-lag.csv

    - Characterizes full growth trajectory including saturation
    - Produces temperature-performance curves (TPCs)

================================================================================
KEY CORRECTION (April 22, 2026)
================================================================================

ISSUE:
  Previous comparison plots extended the growthtools exponential line across
  the entire time course, including the saturation phase where the exponential
  model is invalid. This was misleading about method validity.

SOLUTION:
  Script 12-comparison-logistic-vs-growthtools-CORRECTED.R:
  - Automatically detects exponential phase boundary
  - Truncates growthtools predictions to exponential phase only
  - Clearly shows domain of validity for each method
  - Provides transparent visualization for reproducibility

OUTPUT:
  figures/comparison_logistic_vs_growthtools_42C_CORRECTED.pdf
  figures/comparison_logistic_vs_growthtools_42C_CORRECTED.png

REPRODUCIBILITY NOTES
================================================================================

All scripts are self-contained and documented:
  1. Load data from data-processed/ or data-raw/
  2. Define helper functions
  3. Perform analysis
  4. Save results to data-processed/ and figures/

To reproduce:
  1. Ensure input data files exist
  2. Run scripts in sequence (02, 03/09/10, 12)
  3. Check console output and figures for validation
  4. Compare against saved results

Version Control:
  - Each script includes last updated date
  - CORRECTION-LOG.md documents changes
  - Previous versions (if archived) are noted in comments
